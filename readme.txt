== Description ==

This plugin used to remove the old slugs of post and pages.

= Features =

*  Extremely easy to use

*  Secure

*  Ability to search by Post ID, Post Name and Old Slug. 

*  No Admin Settings.

== Installation ==

1. Upload remove-old-slugs-post-pages folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
